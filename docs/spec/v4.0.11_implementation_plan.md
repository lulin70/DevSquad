## 二、现状分析

### FakeLLMBackend 重复

| 文件 | 行号 | API 差异 |
|------|------|----------|
| `tests/test_feedback_control_loop.py` | L43-60 | `__init__(responses: list[str], default: str)` — 序列响应 |

**方案**: 创建 `scripts/check_dependency_sync.py`，在 CI lint job 中运行。
