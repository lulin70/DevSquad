每个优化任务都必须有回滚计划：

```bash
# 优化前：创建分支 + 打标签

git tag before-optimization-1.1

# 优化中：频繁提交

# 优化后：测试验证

git reset --hard before-optimization-1.1

# 如果测试通过，合并到主分支
```
